# Python Mini Projects

A collection of small, self-contained Python scripts demonstrating common programming tasks and simple project ideas. Each file is a standalone mini-project you can run, modify, and learn from.

## Table of Contents

- **Overview:** Quick summary of this repository.
- **Projects:** Short descriptions and how to run each script.
- **Usage:** Requirements and run examples.
- **Contributing:** How to contribute.
- **License:** Repository license.

## Overview

This repository groups multiple mini-projects intended for learning and quick experimentation. They cover topics such as simulations, simple CRUD-style programs, utilities, and small system models.

## Projects

- `animal_sounds.py`: Example demonstrating mapping animals to their sounds and simple user interaction.
- `atm_simulations.py`: Simulates an ATM interface with basic operations like withdraw, deposit, and balance check.
- `bank_account.py`: A simple bank account class with deposit, withdraw, and balance methods; useful for OOP practice.
- `calculator.py`: Command-line calculator demonstrating basic arithmetic operations and input parsing.
- `employee_salary.py`: Calculates employee salaries, deductions, and net pay — useful for basic payroll logic.
- `food_ordering.py`: Minimal food ordering flow simulating menu selection and order summary.
- `hospital_appointment.py`: Simple appointment booking simulation for a hospital or clinic.
- `hotel_booking.py`: Basic hotel room booking simulation with availability checks.
- `inventory_stock.py`: Tracks product stock levels and supports simple inventory operations.
- `library_system.py`: A tiny library management system for borrowing and returning books.
- `notification_system.py`: Demonstrates simple notification logic (console-based alerts or messages).
- `online_shopping_cart.py`: Shopping cart simulation with add/remove items and checkout summary.
- `payment_gateway.py`: Mock payment processing flow for testing transaction logic.
- `pet_adoption.py`: Simulates a pet adoption process with listings and adoption actions.
- `ride_sharing.py`: A minimal ride-sharing simulation matching riders and drivers.
- `school_management.py`: Basic school management utilities like student lists and class assignments.
- `smart_home.py`: Simulates simple smart home device control scenarios.
- `student_report_card.py`: Generates student report cards and calculates grades/averages.
- `university_management.py`: Higher-level management utilities for courses, students, and enrollment.
- `vehicle_rental.py`: Vehicle rental simulation with booking and return workflows.

> Note: File names map directly to individual scripts; open the script to see usage details and any script-specific instructions.

## Usage

Requirements:

- Python 3.8 or newer

Run a script from the repository root. Examples:

```bash
python animal_sounds.py
python calculator.py
```

Many scripts are interactive and will prompt for input at runtime.
